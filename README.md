# TOMS — The Organised Money System

TOMS is a personal finance dashboard designed to sync bank transaction data, track budgets, forecast spending, and provide daily financial guidance.

## Tech Stack

- React + Vite + TypeScript
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Starling Bank API

## Current Status

In development.

## Planned Features

- User authentication
- Encrypted Starling token storage
- Starling transaction sync
- PostgreSQL transaction storage
- Monthly budget tracking
- Safe daily spend calculation
- Category-based spending breakdown
- Forecasting and insights
